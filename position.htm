Positioning

Elements are positioned using: top | bottom | left | right properties. They work only when position property is set and work differently depending on the position value.

Static: (Default)

Elements are not affected by the top, bottom, left, and right properties.

Relative:

Setting the top, right, bottom, and left properties of a relatively-positioned element will cause it to be adjusted away from its normal position. Other content will not be adjusted to fit into any gap left by the element.

Absolute:

An element is positioned relative to the nearst positioned ancestor. If it has no ancestors, it uses the document body.

Note: Absolute positioned elements are removed from the normal flow, and can overlap elements.

Fixed:

An element is positioned relative to the viewport. It stays in the same place even if the page is scrolled.

Sticky

An element is positioned based on the user's scroll position.

A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).
